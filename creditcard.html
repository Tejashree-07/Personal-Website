CREDIT CARD FRAUD PREDICTION

As part of a graduate-level elective, I collaborated on a team project focused on developing a predictive model to detect credit-card frauds, by using a dataset of over 550,000 credit card transactions. This project involved using predictive models to identify fraudulent behavior, focusing on extracting meaningful patterns and minimizing false alarms.

Time-based and behavioral features were engineered from raw transactional data, and multiple classification models were trained (including Logistic Regression, Naive Bayes, LDA, Random Forest, SVM, and XGBoost), and evaluated using ROC-AUC, F1-score. XGBoost achieved the highest performance (AUC = 0.998, F1 = 0.98). Feature importance was performed to identify transaction category, frequency, and amount as key predictive indicators.

While the application was financial, the methods such as feature engineering and model tuning can directly apply to predictive maintenance, fault detection, and process risk analysis in manufacturing environments.

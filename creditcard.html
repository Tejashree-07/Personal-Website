CREDIT CARD FRAUD PREDICTION

I led a team of three for a course project on predicting credit card frauds in the US, using R.

The project focused on creating machine learning algorithms to predict frauds in the US. A dataset of over 555,000 transactions to identify patterns and features indicative of fraudulent activities was utilized. Key features, such as user age, transaction categories, amounts, and previous fraud occurrences, were engineered to enhance model accuracy.

Several machine learning models, including Naive Bayes, Logistic Regression, Random Forest, Support Vector Machine (SVM), and XGBoost, were implemented and evaluated using metrics such as True Positive Rate (TPR), False Positive Rate (FPR), F1 Score, and AUC. Feature importance analysis was performed to rank the predictors influencing fraud detection.

The project also incorporated visualizations such as histograms of transaction amounts, user ages, and fraud frequencies, choropleth maps highlighting states and cities with the highest fraud rates, and ROC curves to compare model performances.
